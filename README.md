# 🎮 GameStore - Интернет-магазин игр, ключей и аккаунтов

Современный интернет-магазин цифровых товаров (игры, ключи активации, аккаунты, DLC) с полным функционалом e-commerce.

## 🚀 Возможности

### Публичная часть
- **Главная страница** - слайдер с популярными товарами, акции, новинки, быстрый поиск
- **Каталог товаров** - фильтры, сортировка, пагинация, категории
- **Страница товара** - фото, описание, характеристики, отзывы, рейтинг
- **Корзина и оформление заказа** - добавление/удаление товаров, выбор оплаты и доставки
- **Новости** - лента новостей, страница новости
- **Услуги** - бустинг, обучение, запись на услуги
- **Поиск** - по товарам, новостям и услугам
- **Авторизация/Регистрация** - с валидацией и загрузкой аватара
- **Личный кабинет** - история заказов, избранные новости, просмотры

### Админ-панель
- **Дашборд** - статистика, последние заказы, графики
- **Управление товарами** - CRUD товаров с модальными окнами
- **Управление новостями** - создание/редактирование новостей
- **Управление услугами** - добавление услуг и слотов
- **Управление заказами** - смена статусов, просмотр деталей

### Особенности
- 🌓 Темная/светлая тема оформления
- 👁️ Режим для слабовидящих (увеличение)
- 📱 Адаптивный дизайн
- ⚡ Анимации и переходы
- 🔒 Безопасная авторизация
- 📧 Уведомления (flash messages)
- 📊 Валидация форм
- 💾 Все данные из БД MySQL

## 🛠 Технологии

- **Backend**: PHP 8.x (PDO, MVC архитектура)
- **Frontend**: HTML5, CSS3 (CSS Variables), JavaScript (ES6+)
- **База данных**: MySQL/MariaDB
- **Зависимости**: Composer

## 📦 Установка

1. **Клонировать репозиторий:**
```bash
git clone <repository_url>
cd gamestore
```

2. **Установить зависимости:**
```bash
composer install
```

3. **Настроить базу данных:**
```bash
cp .env.example .env
# Отредактируйте .env с вашими данными БД
```

4. **Импортировать БД:**
```bash
mysql -u root -p < database/schema.sql
mysql -u root -p < database/seeds/initial_data.sql
```

5. **Запустить веб-сервер:**
```bash
# Для встроенного сервера PHP
php -S localhost:8000 -t public/
```

## 📊 База данных

Основные таблицы:
- `users` - пользователи
- `products` - товары
- `categories` - категории
- `orders` / `order_items` - заказы
- `news` - новости
- `services` - услуги
- `reviews` - отзывы
- `cart` - корзина
- `favorites` - избранное
